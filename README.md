# SMART-EWS

### Smart Early Warning System for Students

SMART-EWS is a full-stack web application designed to identify and monitor students who may be academically or behaviorally at risk. The system helps faculty, counselors, and administrators take early intervention actions.

---

## 🚀 Tech Stack

Frontend

* React
* TypeScript
* Vite
* TailwindCSS

Backend

* Node.js
* Express.js

Database

* MongoDB

Authentication

* JWT Authentication

---

## 📊 Features

* Multi-role authentication (Student, Faculty, Counselor, Admin)
* Student risk prediction system
* Academic performance tracking
* Behavioral rating system
* Counselor intervention tracking
* Admin analytics dashboard

## ⚙️ Installation

Clone the repo

```
git clone https://github.com/yatri28/smart-EWS.git
```

Install backend dependencies

```
cd server
npm install
npm run dev
```

Install frontend dependencies

```
cd client
npm install
npm run dev
```
